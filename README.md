# Hangman-Game

A simple Hangman game implemented in **Java**.  
Fully developed by me, except for the ASCII art, which was sourced from [fssnip](https://www.fssnip.net/mO/title/Hangman).

---

## Features

- Classic Hangman gameplay  
- 2-player mode  
- Displays ASCII art for each stage of the game  
- User-friendly command-line interface  

---

## How to Play

1. Run the program in a Java IDE or from the command line.  
2. Player 1 creates a word.  
3. Player 2 tries to guess the word. After each incorrect guess, a part of the hangman is drawn.  
4. Win by guessing the word before the hangman is fully drawn.
